Compile the Java Files
Ensure all Java files (Main.java, MatrixModel.java, MatrixView.java, and MatrixController.java) are in the same directory. Then, open a terminal or command prompt and navigate to that directory. 2. Run the Program

After successful compilation, execute the program


Interact with the GUI
Enter the number of rows and columns for Matrix A and Matrix B.
Click "Create Matrices" to open matrix input windows.
Enter matrix values manually or load them from a file using "Open".
Click "Save" to store matrix data.
Select an operation (Add, Subtract, Multiply, Transpose) from the dropdown.
Click "Execute Operation" to perform the calculation.
View the status updates for errors or successful execution.
Use "Reset" to clear inputs or "Quit" to exit the application.
